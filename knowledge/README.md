# Carpeta de Conocimiento

Coloca aquí los documentos que SmartBot debe poder consultar:

- **Archivos soportados**: `.pdf`, `.md`, `.txt`
- **Organización**: Puedes crear subcarpetas por departamento/tema
- **Reindexar**: Después de agregar/modificar archivos, usa el botón "Reindexar" en la app o reinicia el servidor

## Ejemplos de documentos útiles:
- Políticas internas (vacaciones, viáticos, código de conducta)
- SOPs por departamento
- Manuales de herramientas
- FAQ internas
- Organigramas
- Procesos de onboarding/offboarding
